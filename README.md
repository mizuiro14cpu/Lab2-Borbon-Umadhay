# Lab2-Borbon-Umadhay
A game library management system built with Express that performs CRUD operations and uses Supabase as its backend

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/users` | Create a new user |
| GET | `/api/users` | Get all users |
| POST | `/api/games` | Create a new game entry |
| GET | `/api/games` | Get all registered games |

## Installation and Setup
1. Clone the repo:
```bash
git clone https://github.com/mizuiro14cpu/Lab2-Borbon-Umadhay.git
cd Lab2-Borbon-Umadhay
```
2. Install dependencies:
```bash
npm install
```
3. Create a .env and .env.test file:
```bash
#.env
SUPABASE_URL=https://your-project-url.supabase.co
SUPABASE_KEY=your-publishable-key
PORT=3000

#.env.test
SUPABASE_URL=https://your-project-url.supabase.co
SUPABASE_KEY=your-secret-key
```
